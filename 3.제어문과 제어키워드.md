```
● 제어문
-if 제어문(선택 제어문)
if(조건식){  //true면 실행됨

    실행 내용;
}
else{    // if에서 조건식의 나머지 부분

    실행내용
}

- switch 제어문

switch(점프위치변수)  { //->정수, 문자, 문자열 

case 위치값 1:
   실행 내용;

case 위치값 2:
   실행 내용;

case 위치값 n:
   실행 내용;

default:
   실행 내용;
}

- for 제어문(반복문)
for(초기식; 조건식; 증감식) {
    실행 내용;
}

- while 제어문(반복문)
초기식;
while(조건식) {

    실행내용;  //조건식이 true인 동안 실행
    증감식;
}

- do-while 제어문(반목문)
초기식;
do {
    실행내용;
    증각식;  // 최초 1회는 무조건 실행

} 
while(조건식); 

-break
: switch 문이나 반복문 (for, while, do-while) 안에서 사용
: 문장이 실행되면 가장 가까운 반복문이나 switch 문을 종료하고, 해당 반복문 밖으로 제어를 전달

-continue
: 복문 안에서 사용되며, 이 문이 실행되면 현재의 반복을 중지하고 다음 반복으로 넘어감
```
